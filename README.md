# Similarities-Between-Text-Files
## Summary 
Creates a matrix of similarities between contents of text files using the cosine similarity formula. 

## Dependencies
- python: 3.6.5
- pandas: 0.25.3
- numpy: 1.14.3
- nltk: 3.3.0
- scikit-learn: 0.19.1
- math
- string
